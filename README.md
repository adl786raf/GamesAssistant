**A lightweight, open-source strategy and tracking tool for Linux.**

GamesAssistant is a Python-based utility designed to help track patterns, manage strategies, and provide audio-visual feedback during gaming or probability tasks. 

## âœ¨ Features
* **Two Core Tracking Modes:**
  * **Static Mode:** Manually log events and track fixed sequences.
  * **Time-Based Mode:** Track entries against a countdown or specific time intervals.
* **Visual Themes:** Clean, custom Tkinter GUI.
* **Audio Feedback:** Distinct sound alerts for Win/Loss conditions.
* **Lightweight:** Built for Linux, fully functional on older 32-bit (i686) systems and scalable to 64-bit architectures.
